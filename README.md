# Java Lombok Car Demo
A simple Java project demonstrating the use of Lombok annotations with a `Car` model and a demo class.
The project shows how Lombok can automatically generate boilerplate code such as getters, setters, constructors, `equals()`, `hashCode()`, and `toString()`, making the code cleaner and easier to maintain.
